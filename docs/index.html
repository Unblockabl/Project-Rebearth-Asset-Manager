Welcome to the Project Rebearth Asset Manager

Project Rebearth Asset Manager is a resource pack manager/loader for Project Rebearth

To use, download the program and place it in the same location as the ProjectRebearthGame.exe file. To find this, open Steam, right-click on 'Project Rebearth', select 'Manage', select 'Browse Local Files'. In the File Explorer window, extract the zip from the downloads page there. Run 'asset_manager.exe'. In the text field, paste your resource pack then click apply. Restart Project Rebearth and it is now installed! To get the normal assets, please press the 'Reset Pack' button! For more detailed instructions and how to create a custom pack, visit this link.
